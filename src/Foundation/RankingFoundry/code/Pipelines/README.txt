UnicornSyncStart - Occurs when a sync starts - which can be multiple configurations
UnicornSyncBegin - Occurs when a configuration begins syncing
UnicornSyncComplete - Occurs when a configuration completes syncing
UnicornSyncEnd - Occurs when a sync operation - which can be multiple configurations - completes
unicornReserializeStart - Occurs when a reserialize operation begins - which can be multiple configurations
UnicornReserializeComplete - Occurs when a configuration has been reserialized successfully
UnicornExpandConfigurationVariables - Occurs when loading configurations XML, used to replace variable expressions in config definitions
